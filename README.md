# Efficient use of the Grace Hopper GPU nodes on Dardel and Arrhenius

These pages contain information about the training event on 30-31 March 2026, given under NAISS and PDC.  The lectures are provided by staff from NVIDIA and Hewlett Packard Enterprise (HPE)

## Location

Kungliga Tekniska Högskolan (KTH), Stockholm. 

## Time and Date

- 30-31 March 2026
- 9:00 - 17:00 each day. 

## Registration 

Registration is set to close on 18th March 2026.  Acceptance on the course will be announced no later than the 23rd March 2026. 

[Registration will be available on the main event page on naiss-training.github.io](https://naiss-training.github.io/Using-Grace-Hopper/#registration)
